# Курс ЦБ РФ · Онлайн

Веб-сайт для отслеживания официального курса валют Центрального банка РФ:
таблица курсов с изменением день-к-дню и график истории за 7/30/90 дней.

Данные берутся напрямую с `cbr.ru` (`XML_dynamic.asp`), без сторонних API.

## Стек

- Backend: Python 3 + Flask
- Frontend: vanilla HTML/CSS/JS (SVG-график без внешних библиотек)

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Сайт будет доступен на http://127.0.0.1:5000/.

## Структура проекта

```
app.py               # Flask-бэкенд: получение и кэширование курсов ЦБ РФ
templates/index.html # разметка страницы
static/style.css     # стили (светлая/тёмная тема)
static/app.js        # таблица курсов, опрос API, отрисовка графика
requirements.txt      # зависимости Python
```

## API

- `GET /api/rates` — текущий курс и изменение за день по каждой валюте.
- `GET /api/history/<code>?days=30` — история курса валюты (`code`: USD, EUR, GBP, CNY, JPY, CHF, TRY, KZT, BYN, AMD, INR, AED; `days`: 7–90).

## Примечания

- ЦБ РФ публикует официальный курс раз в рабочий день; страница опрашивает
  сервер каждые 60 секунд, но сам курс меняется не чаще, чем обновляет ЦБ.
- Ответы `/api/rates` и `/api/history` кэшируются на сервере на 10 минут,
  чтобы не перегружать `cbr.ru` запросами.
